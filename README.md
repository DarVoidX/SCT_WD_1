## Features & Technologies
- **Responsive Layout**: Designed for mobile, tablet, and desktop viewports.
- **Fixed & Dynamic Navbar**: Sticky navigation bar that switches background styles on scroll.
- **Modern UI/UX**: Hover transitions and custom gradient backgrounds.
- **Tech Stack**: HTML5, CSS3 (Flexbox & Media Queries), Vanilla JavaScript.
